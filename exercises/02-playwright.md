# 02 · Dale un navegador

## Descubre

[Playwright MCP oficial](https://github.com/microsoft/playwright-mcp)

## Construye

Pide a tu agente:

> Instala y configura Playwright MCP oficial en este proyecto siguiendo su documentación actual. Después mejora nuestra skill para explorar rutas, completar procesos, probar teclado, observar el foco y conservar evidencia. Verifica que el MCP funciona antes de auditar.

## Prueba

Vuelve a ejecutar el agente contra `TRAINING_URL_1` y después contra:

`TRAINING_URL_2`

## Discute

- ¿Qué encontró ahora que antes era invisible?
- ¿Probó estados o solo la carga inicial?
- ¿Confundió un snapshot con un lector de pantalla?

Modificad la skill con una regla concreta para decidir cuándo explorar, cuándo evaluar y cuándo detenerse.
