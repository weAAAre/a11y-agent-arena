# 02 · Dale un navegador

## Descubre

[Playwright MCP oficial](https://github.com/microsoft/playwright-mcp)

## Construye

Pide a tu agente:

> Instala y configura Playwright MCP oficial en el entorno del agente que hemos elegido, siguiendo la documentación actual de ambos. Después mejora nuestra skill para explorar rutas, completar procesos, probar teclado, observar el foco y conservar evidencia. Verifica con una navegación real que la integración funciona antes de auditar.

## Prueba

Vuelve a ejecutar el agente contra
<https://a11y-agent-arena-challenges.vercel.app/training-1/> y después contra:

<https://a11y-agent-arena-challenges.vercel.app/training-2/>

## Discute

- ¿Qué encontró ahora que antes era invisible?
- ¿Probó estados o solo la carga inicial?
- ¿Confundió un snapshot con un lector de pantalla?

Modificad la skill con una regla concreta para decidir cuándo explorar, cuándo evaluar y cuándo detenerse.
